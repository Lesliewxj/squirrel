# 校园闲置物品交易网站


基于SSM技术简易校园二手市场，主要的功能有：二手商品发布、商品列表展示、用户注册登录、收藏商品、商品评论、购买商品、管理后台主要是:用户信息管理、商品信息管理、订单管理、钱包管理、系统设置等。

运行环境

jdk8 + tomcat7 + mysql5.6 + IntelliJ IDEA（Eclipse也可以）

项目技术

spring+spring mvc+mybatis+bootstrap+jquery
